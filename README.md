# Graph Algorithms: TSP and Bellman-Ford

A C++ project for implementing and analyzing classic graph algorithms, including Bellman-Ford shortest path and Traveling Salesman Problem optimization.

## Features

- Represent weighted directed graphs using edge lists
- Compute shortest path values using Bellman-Ford
- Reconstruct shortest paths between two vertices
- Solve the Traveling Salesman Problem for small graphs using dynamic programming
- Use heuristic/backtracking logic for larger graph instances
- Demonstrate graph traversal, path reconstruction, and optimization techniques
